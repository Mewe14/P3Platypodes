Learn Bubble Sort and Binary Search

Insert 5 unique numbers and pick one of those numbers to search for

Bubble Sort

Purpose: The bubble sort will take your 5 numbers and sort them from low to high

First the sort takes the first two numbers in the list and evaluates which value is larger, swapping if necessary

The sort will continue to compare the congruent values until the list is ordered from low to high

Eventually, we get the final list as observed here: {{nums.list}}

Binary Search

Purpose: The binary search will find the number you select

Binary Searches require an ordered list, so the search will utilize the list that was made by the bubble sort

First the search will find the middle value

Then the search will evaluate if the desired value is higher, lower, or equal to the middle value

From here, the search will continue looking at the numbers right of the original value and establish a new middle value

The repetition continues to whittle down the selection pool until the desired value is found

Congrats the desired value is found in position {{nums.index}}
